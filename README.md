Telegram-Бот Бухгалтер    @accountant_b_bot   
Интеллектуальный ассистент по бухгалтерскому и налоговому учету в Республике Казахстан

Этот проект представляет собой Telegram-бота, разработанного на Python, который использует большие языковые модели (LLM) OpenAI и технологию RAG (Retrieval-Augmented Generation) для предоставления точных консультаций на основе актуального Налогового кодекса РК. Бот специально настроен для помощи индивидуальным предпринимателям (ИП) и малому бизнесу, работающему в Казахстане.

Основные возможности:
Консультации по НК РК: Предоставление точных и актуальных ответов по налогам, отчетности и срокам на основании загруженной базы знаний.

Управление контекстом: Бот запоминает, какую компанию (ТОО, ИП) и какой налоговый режим (ОУР, Упрощенка) обсуждает пользователь, и учитывает эту информацию в ответах.

RAG-технология: Использование векторной базы данных FAISS для поиска наиболее релевантных статей Налогового кодекса, что обеспечивает точность ответов.

История диалога: История сообщений сохраняется в базе данных SQLite, позволяя пользователю продолжить прерванный разговор.
